# 🔧 Фиксики — Приложение мастеров

Семейное приложение для мастеров по ремонту бытовой техники.

## Что внутри

- 📋 **Задачи** — расписание на день, статусы, навигатор
- 👥 **Клиенты** — база клиентов, история визитов, фото
- 📸 **AI-Сканер** — распознавание шильдиков, деталей, кодов ошибок
- 🤝 **Партнёры** — доверенные мастера, передача клиентов
- 💰 **Кошелёк** — доходы и расходы
- 👑 **Админка** — для Николая (видит всех)

1. Откройте ссылку в Chrome (Android) или Safari (iPhone)

3. Safari: □↑ → «На экран Домой»
